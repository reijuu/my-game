■ ゲーム紹介
※本作はまだ開発段階です。製品版ではさらに多くの要素が追加されます。

RPGツクール歴一週間ですが、やってくれるとうれしいです。

自由にスキルを習得し、自分だけのビルドを構築できるRPG。
レベル上限の概念に捕らわれない、好きなだけ育成できる作品です。
